# HyPrism Website

[![Website](https://img.shields.io/badge/Live-yyyumeniku.github.io/hyprism--site-207e5c?style=for-the-badge)](https://uyuyantavukv2.github.io/Tavuktale-site/)

Simple landing page for Tavuktale launcher with real-time GitHub contributors and download counter.

## Features

- Real-time download counter from GitHub Releases API
- Dynamic contributor list with avatars
- Direct download links for Windows, macOS, and Linux
- Green theme matching Tavuktale's design
